## Usage


### Run development server

```bash
yarn
yarn dev
```

### Export static files

```bash
yarn export
```

### Run Storybook

```bash
yarn storybook
```

### Build static Storybook

```bash
yarn build-storybook
```

### Test the export

```bash
[yarn global add serve]
cd out
serve -p 8080
```
