## Usage

### Run development server

```bash
yarn
yarn develop
```

### Export static files

```bash
yarn build
```

### Run Storybook

```bash
yarn storybook
```

### Build static Storybook

```bash
yarn build-storybook
```

### Serve the build

```bash
yarn serve
```
