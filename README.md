# Website with Next.js

## Usage

```bash
yarn
yarn dev
```

## Run Storybook

```bash
yarn storybook
```

## Build Static Storybook

```bash
yarn build-storybook
```

## Build static

```bash
next build
next export
```

Test the static deployment:

```bash
cd out
[yarn global add serve]
serve -p 8080
```
